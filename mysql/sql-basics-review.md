# SQL基础复习笔记

## 高级数据过滤
### Where子句
#### 四种操作符
* AND操作符
* OR操作符
* IN操作符：和OR操作符能完成相同的功能，并且语法比OR操作符更家清晰。IN操作符的执行速度也一般比OR执行更快。并且，最重要的是能够**嵌套包含其他SELECT子句**
* NOT操作符：WHERE子句中NOT操作符用来否定其后面条件的关键字。
  ```SQL
  select name,email from USER where id not in (1,2,3)
  ```
